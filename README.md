<br>
### User Story
* 串接 API 並可以呈現產品列表、加入購物車、購物車列表功能
* 表單送出前進行表單驗證（必要完成），驗證內容包含：
    * 姓名：必填
    * Email：須符合格式
    * 電話：必填，超過 8 碼，input type 為 tel
    * 地址：必填
    * 付款方式：WebATM、ATM、Barcode、Credit、ApplePay、GooglePay
    * 留言：非必填
